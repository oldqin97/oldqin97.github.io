    instanceof 的实现原理

    const newInstance = (left, right) => {
  if (typeof left !== 'object' && typeof left === null) false;

  let proto = Object.getPrototypeOf(left);

  // 沿着原型链去找,直到找到相同的原型对象
  while (true) {
    if (proto === null) return false;
    if (proto === right.prototype) return true;
    proto = Object.getPrototypeOf(proto);
  }
};

    1. 要求检测的实例必须是对象类型

    # 实用的检测方式, Object.prototype.toString

    每一种数据结构的构造函数的原型上都有 toString 方法,除了 Object.prototype 上的 toString 是用来返回当前实例所属类的信息（检测数据类型的），其余的都是转换为字符串的, 改变 toSting 方法检测的 this 值, 让它返回详细内容

    const getType = target => {
  return Object.prototype.toString.call(target).slice(8, -1).toLowerCase();
};
