    较为陌生的对象

    oldqin 4/4/2021 es3

    对象是现实的抽象方式, JavaScript 中一切对象皆是对象, JavaScript 中的对象被设计成一组属性的无序集合, key 和 value 组成, key 是一个标识符名称, value 可以是任意类型

    # Object.defineProperty(定义对象, 定义对象的 key, 数据/存取属性描述符) 方法会直接在一个对象上定义一个新属性, 或者修改一个对象的现有属性, 并返回该对象.

    数据属性描述符

    • [[configurable]]: 表示属性是否可以通过 delete 删除属性, 是否可以修改它的特性配置项,或者是否可以将它修改为存取属性描述符
    • [[enumerable]]: 表示属性是否可以同过 for-in 或者 Object.keys() 返回该属性
    • [[writeable]]: 表示是否可以修改属性的值
    • [[value]]: 属性的 value 值, 读取属性返回该值, 修改属性会对其修改 default === undefined
    'use strict';
const info = {
  nickname: 'qwe',
  grade: 1,
};
Object.defineProperty(info, 'address', {
  value: 'gz', // default value undefined
  // 该属性不可删除/也不可以重新定义属性的描述符 default value false
  configurable: false,
  // 该属性是配置对应属性是否可以枚举的 default value false
  enumerable: false,
  // 该属性是是否可以写入值的 default value false
  writable: false,
});

// configurable : false 不可删除/ 不可修改配置项
delete obj.address; // !  Cannot delete property 'address'

Object.defineProperty(obj, 'address', {
  enumerable: true, // ! can't redefine property: address
});

// + enumerable 不可枚举
for (var key in obj) {
  console.log(key);
}
console.log(Object.keys(obj)); // 没有address 遍历项目

// ! 可以通过getOwnPropertyNames 遍历出key值
console.log(obj[Object.getOwnPropertyNames(obj)[2]]);

// + writeable 不可写入
obj.address = 'hz';
console.log(obj.address); // gz

    # 通过属性描述符来看三个函数 Object.freeze() / Object.seal() 这三个函数本质上的作用就是通过遍历的方式修改对象内全部属性的 属性描述符

    Object.freeze 使对象的内属性 不可删除,不可配置,不可写入

    const test = { t: 1 };
Object.freeze(test);
console.log(Object.getOwnPropertyDescriptor(test, 't'));

{
  configurable: false;
  writable: false;
  enumerable: true;
  value: 1;
}
