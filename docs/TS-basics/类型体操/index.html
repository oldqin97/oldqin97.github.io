类型体操

条件类型​

开发中,我们需要基于输入的值来决定输出的值,同样我们也需要基于输入的值的类型来决定输出的值的类型

条件类型 就是用来帮助我们描述输入类型和输出类型之间的关系

  • 条件类型的写法有点类型于 JS 中的条件表达式 (condition ? trueExpression : falseExpression)

SomeType extends OtherType ? TrueType : FalseType;

type IDType = number | string;

type ResType = number extends IDType ? true : false;
// ResType true

案例​

function sum(num1: number, num2: number): number;
function sum(num1: string, num2: string): string;
function sum(num1: any, num2: any) {
  return num1 + num2;
}
function sum<T>(num1: T, num2: T): T extends number ? number : string;
function sum(num1: any, num2: any) {
  return num1 + num2;
}

let res = sum(1, 312);
let res1 = sum('qwe', 'ewq');

在条件类型中推断 (infer)​

条件类型提供了 infer 关键字, 可以从正在比较的类型中推断类型, 然后在 true 分支中引用该推断结果

我们现在有个函数类型,想要获取到一个函数的参数类型和返回值类型

type CalcFnType = (num1: number, num2: number) => number;

type ReturnType<T extends (...args: any[]) => any> = T extends (...args: any[]) => infer R
  ? R
  : never;

// 内置工具
type CalcReturnType0 = ReturnType<CalcFnType>; // number
type fooReturnType0 = ReturnType<typeof foo>; // string

type CalcReturnType = ReturnType<CalcFnType>; // type CalcReturnType = number

type Parameters<T extends (...args: any[]) => any> = T extends (...args: infer P) => any
  ? P
  : never;

type CalcParameterType = Parameters<CalcFnType>; // type CalcParameterType = [num1: number, num2: number]

分发条件类型​

当我们在泛型中使用条件类型的时候, 如果传入一个联合类型,就会变成分发的 (distributive)

type ToArray<T> = T extends any ? T[] : never;

type newType = ToArray<number | string>;
// type newType = string[] | number[]

内置工具重写​

Required\<Type>​

用于构造一个 Type,下面所有的类型都为必填的类型

type MyRequired<T> = {
  [K in keyof T]-?: T[K];
};

interface IPerson {
  name: string;
  grade?: number;
  test?: string;
}

type IPersonRequired = MyRequired<IPerson>;
/**
  type IPersonRequired = {
    name: string;
    grade: number;
    test: string;
  }
*/

type res = Required<IPerson>;

Readonly\<Type>​

用于构造一个 Type 内所有属性全都设置为只读类型

type MyReadonly<T> = {
  readonly [K in keyof T]: T[K];
};

interface IPerson {
  name: string;
  grade: number;
  test: string;
}

type IPersonReadonly = MyReadonly<IPerson>;

/**
 type IPersonReadonly = {
    readonly name: string;
    readonly grade: number;
    readonly test: string;
  }
 */

type IPersonReadonly = Readonly<IPerson>;

Record\<Key,Type>​

用于构造一个对象类型,它所有的 key 都是 Keys 类型,它所有的 value 都是 Type 类型

interface IPerson {
  name: string;
  grade: number;
  test: string;
}

type MyRecord<K extends keyof any, T> = {
  [P in K]: T;
};

const p1: IPerson = {
  name: '',
  grade: 0,
  test: '',
};
const p2: IPerson = {
  name: '',
  grade: 0,
  test: '',
};
type CityType = '贵州' | '北京';

const data: MyRecord<CityType, IPerson> = {
  贵州: p1,
  北京: p2,
};

const data2: Record<CityType, IPerson> = {
  贵州: p1,
  北京: p2,
};
